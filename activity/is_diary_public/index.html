Write a C program, is_diary_public.c, which prints 1 if a file named $HOME/.diary exists and is publically readable (readable to all users).

It should print 0 otherwise.

By $HOME we mean the value of the environment variable HOME. 

dcc is_diary_public.c -o is_diary_public
echo >$HOME/.diary
chmod 644 $HOME/.diary
./is_diary_public
1
chmod 600  $HOME/.diary
./is_diary_public
0
rm $HOME/.diary
./is_diary_public
0
You can assume if $HOME/.diary exists its is an ordinary file.

No error checking is required.

Your program can assume it is always a single argument.

It should always one line of output. The line of output should contain only 0 or 1.

Your solution must be in C only.

You are not permitted to run external programs. You are not permitted to use system, popen, posix_spawn, fork or exec.
